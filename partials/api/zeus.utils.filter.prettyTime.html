prettyTime
filter in module zeus.utils

Description

Converts a date represented by numeric epoch to a human readable string.

Usage

In HTML Template Binding

{{ prettyTime_expression | prettyTime }}

In JavaScript

$filter('prettyTime')(timeInMillis)

Parameters

ParamTypeDetails
timeInMillis number

Date as numeric epoch( in milliseconds )

Returns

string

The date as a human readable string
