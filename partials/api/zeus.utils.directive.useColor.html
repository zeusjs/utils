useColor
directive in module zeus.utils

Description

A directive that assigns a color value to a particular attribute of a directive

Dependencies

  • Colors

Usage

as attribute
<ANY use-color="{object}">
   ...
</ANY>

Parameters

ParamTypeDetails
useColorobject

An object literal containing the attribute name (key) to which a color (value) may be assigned
