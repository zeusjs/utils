timeAgo
filter in module zeus.utils

Description

Provides human readable string representing the time elapsed since the event occurred.

Usage

In HTML Template Binding

{{ timeAgo_expression | timeAgo[:refTime] }}

In JavaScript

$filter('timeAgo')(eventTime[, refTime])

Parameters

ParamTypeDetails
eventTime number

Time when event occurred in millisec

refTime
(optional)
 number

Reference time

Returns

string

Human readable format of the event time
