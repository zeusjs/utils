bytes
filter in module zeus.utils

Description

Converts a number bytes to a string kilo/mega/giga/tera or peta bytes

Usage

In HTML Template Binding

{{ bytes_expression | bytes[:precision] }}

In JavaScript

$filter('bytes')(size[, precision])

Parameters

ParamTypeDetails
size number

Size in bytes

precision
(optional)
 number

Number of decimals to round off to

Returns

string

Bytes represented as kilo/mega/giga/tera or peta bytes
